
No SQL, o *MERGE* é um comando DML. É uma instrução única que permite realizar *INSERT, UPDATE e DELETE* simultaneamente em uma tabela de destino, baseando-se nos resultados de uma comparação com uma tabela de origem.

Aqui, ele é executado no motor do banco (SQL Server, Oracle, etc.) e atua diretamente sobre dados persistidos e modifica as tabelas. Ele altera fisicamente os dados da tabela (altera seu conteúdo, não seu objeto).

Em termos de performance ele é muito eficiente, porque o banco percorre os dados uma única vez para decidir o que fazer com cada linha.

1. **Sintaxe Básica:** 
	* *Match found? -> UPDATE*
	* *No match found? -> INSERT*
	* *Match only in targer? -> DELETE (opcional)*
