
A arquitetura medalhão é o padrão de design de dados mais utilizado atualmente em Engenharia. Cada camada representa um estágio de tratamento.

* **Camada Bronze (Raw / Dados Brutos):** é o ponto de entrada. Os dados chegam exatamente como estão na fonte original (APIs, bancos de dados, arquivos JSON/CSV, etc.). Neste momento os dados estão sujos, com duplicatas, nomes estranhos e sem tratamento. Aqui, *nada se deleta*, pois é o backup histórico.

* **Camada Silver (Cleansed / Dados Limpos):** é onde ocorre a "limpeza", de forma *moderada*. Aqui os dados são validados, filtrados e padronizados. Neste momento, removemos duplicatas, tratamos valores nulos, convertemos tipos de dados, unimos tabelas simples, etc. Aqui, o dado está pronto para ser usado por analistas de dados.

* **Camada Gold (Curated / Dados de Negócio):** é a camada final, organizada por "assunto", é a camada final para transformações de dados e apresentação em projetos como análise de clientes, análise de estoque, etc. Ela é focada em atender necessidades específicas de negócio.
