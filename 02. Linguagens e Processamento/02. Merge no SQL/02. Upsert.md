
O termo upsert é uma aglutinação de **UP**date + in**SERT**.

Diferente do *merge* (que é um comando padrão SQL), o *UPSERT* é mais um conceito ou uma implementação específica de alguns bancos.

* **Conceito:** "tente inserir este registro. Se a chave primária já existir, em vez de dar erro, atualize os valores existentes."
