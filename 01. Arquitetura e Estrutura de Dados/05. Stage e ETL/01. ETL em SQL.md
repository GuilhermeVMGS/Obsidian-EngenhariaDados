É o processo de utilizar a linguagem SQL para extrair, transformar e carregar dados entre diferentes tabelas ou bancos de dados. 

**E - Extract:** aqui, o SQL é usado para "ler" os dados onde eles estão originalmente. Seu objetivo é trazer o dado para a memória do banco de dados para que ele possa ser manipulado.
**T - Transform:** essa é a parte onde levamos mais tempo. Aqui as regras de negócio são aplicadas usando o funções SQL. Limpamos, convertemos tipos, criamos cálculos, agrupamos, etc.
**L - Load:** é o ato final de salvar o dado transformado na tabela de destino.
