
Antigamente, para consultar dados no Hadoop, você precisava escrever códigos complexos em Java. O Hive surgiu para facilitar isso.

Ele é uma camada que permite que usemos SQL para consultar dados que estão guardados dentro do Hadoop. 

Ele também criou o conceito de DW sobre o DL, permitindo que analistas que só sabem SQL consigam trabalhar com Big Data.
